• DOCTYPE:

    • Um DocType tem função primordial de explicar ao navegador como ele deve ler a página.
    Antigamente existiam vários DocTypes diferentes como por exemplo:

    • HTML 4.01 Strict
    • HTML 4.01 Transitional
    • HTML 4.01 Frameset
    • XHTML 1.0 Strict
    • XHTML 1.0 Transitional
    • XHTML 1.0 Frameset
    • XHTML 1.1

    Aqui está exibida na pratica a forma em que era preciso explicar para o navegador como ver a página com o DocType escolhido:

    <!DOCTYPE HTML PUBLIC "-//W3C//DTD HTML 4.01//EN" "http://www.w3.org/TR/html4/strict.dtd">

    Hoje, com a implementação do HTML 5 tudo ficou muito mais simples na hora de indicar ao navegador de qual forma ele vai ler a página:

    <!DOCTYPE html>
