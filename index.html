Sintaxe HTML

  • ESTRUTURA MÍNIMA:

    • A estrutura mínima de uma página em HTML 5 é a seguinte:

    1. O DocType;
    2. O elemento <html> que determina o pacote de programação/stack html;
    3. O atributo lang= “pt-br” – que posto como atributo ao <html> determina a língua utilizada na página;
    4. Os Elementos <head> e <body>;
    5. A tag <title> dentro do elemento <head> que determina o título da página;
    6. A tag <meta> dentro do elemento <head> com o atributo charset = “utf-8” que define a condição de caracteres de uma página;

  • TEXTOS

    • São a base principal de uma página, ou seja, o conteúdo dela.

    Para títulos podemos utilizar as tags <h1> - <h6> que em ordem crescente vão diminuindo o tamanho dos títulos.
    Uma página deve conter pelo menos uma tag <h1> (princípio de SEO)
    Para parágrafos podemos utilizar a tag <p>;

    No HTML temos os chamados Elementos em Bloco e Elementos em Linha, resumidamente, Elementos em Bloco ocupam a largura total da página e os Elementos em Linha que podem ser postos tanto um ao lado do outro como até mesmo dentro de alguns Elementos em Bloco.

    Aqui alguns exemplos de Elementos em Bloco:

    • <div> (division, ou divisão, tag bloco genérica)
    • <h1> até <h6> (títulos)
    • <p> (parágrafo)
    • <blockquote> (citação em bloco)
    • <b> (põe o bloco em negrito)
    • <i> (põe o bloco em itálico)
    • <ul> (lista não ordenada)
    • <ol> (lista ordenada)
    • <form> (para formulários, veremos mais na unidade "Trabalhando com Formulários")
    • etc.

    E aqui alguns exemplos de Elemento em Linha:

    • <span> (tag em linha genérica)
    • <strong> (põe a linha em negrito)
    • <em> (põe a linha em itálico)
    • <sup> (SuperScript A2)
    • <sub> (SubScript B8)
    • <a> (âncora, usada para links)
    • <img> (imagem)
    • etc.

  • LINKS

    • os links são um ponte entre um site ou outro, no html a tag <a> é utilizada para aplicar tais links no texto.

    Dentro da tag <a> utilizamos o atributo href para indicar a referência do hiperlink.

    E depois acrescentamos o conteúdo do link ou o "nome explicativo" dele.

    Como por exemplo no código abaixo:

    <a href="Referencia.html">Seu Link</a>

    Que gera o seguinte resultado: Seu Link
