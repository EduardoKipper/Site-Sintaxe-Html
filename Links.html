• ÂNCORAS E LINKS

    • Âncoras e Links na verdade são a mesma coisa, porém com nomenclaturas diferentes.

    Sua função é levar o usuário até uma outra página com apenas um "click", porém mesmo sendo uma função simples, contém uma grande variedade de conteúdos que podemos explorar, como:


    Abrindo o Link em uma nova página:

    Para abrir um link que direcione para uma nova aba acrescentamos o atributo/valor target= "_blank".

    <a href="../ancoras-links/" target="_blank" >link real</a>

    o valor _blank sempre irá abrir uma nova página, para que possa (re)utilizar a mesma aba, podemos trocar ele por qualquer valor (nome) que desejarmos como por exemplo target= "tab1".


    Imágens como links:

    Também podemos utilizar de imagens dentro do conteúdo de um link, fazendo que toda a imagem direcione o usuário até uma nova página, como podemos ver no código abaixo:

    <a href="../ancoras-links/">
            <img src="/path/to/imgs/imagem.jpg" />
        </a>
